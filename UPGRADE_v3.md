## 🔧 配置选项

### plugin-config.json（可选）
```json
{
  "mcpUrl": "http://localhost:18060/mcp",
  "autoStartService": true,
  "autoLogin": true,
  "defaultTags": ["AI", "科技", "创业"],
  "maxTitleLength": 20,
  "maxContentLength": 1000,
  "autoRetry": true,
  "maxRetries": 3
}
```

## ⚠️ 错误处理

### 常见错误及解决方案

| 错误 | 原因 | 自动处理 |
|------|------|----------|
| 标题超过20字 | 标题太长 | ✅ 自动缩短 |
| 内容超过1000字 | 内容太长 | ✅ 自动截断 |
| 图片不存在 | 路径错误 | ✅ 自动搜索 |
| 未登录 | Cookies过期 | ✅ 引导重新登录 |
| 发布失败 | 网络问题 | ✅ 自动重试3次 |
| MCP服务未运行 | 服务停止 | ✅ 提示启动命令 |
